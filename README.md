# graySystem implemented using angularJS + beego


## beego Installation
 ```
  go get github.com/astaxie/beego
  go get github.com/beego/bee
  
  export PATH=$PATH:$GOPATH/bin
 ```
    

## gray run

```
cd $GOPATH/src/gray-beego
bee run
```

## Usage

```
http://localhost:9001/v1/grey_configs/1
```
